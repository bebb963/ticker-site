⚠ Sem accent colors. A paleta do Kajo Studio não usa nenhuma cor de destaque (azul, verde, laranja, etc). Toda hierarquia visual é criada por tamanho tipográfico, peso e espaçamento. Se você precisar de alguma cor de acento para o seu projeto, escolha uma única e use-a com extrema parcimônia.

(06 · Efeitos)

Efeitos & Transições

O design tem poucos efeitos mas cada um é preciso. A filosofia é a mesma das cores: minimalismo. Nenhuma animação gratuita — cada efeito serve a navegação ou hierarquia.

Efeito principal — Sticky Scroll (Hero)
Imagem sticky com conteúdo scrollable
Onde: Section Hero · Coluna esquerda
/* Coluna esquerda — fica fixada */
.hero-image-col {
  position: sticky;
  top: 0;
  height: 100vh;            /* Tela inteira */
  overflow: hidden;
}

/* Coluna direita — desce normalmente */
.hero-content-col {
  position: relative;        /* Fluxo normal */
  z-index: 1;
}
A imagem literalmente fica parada enquanto todo o conteúdo da direita (about, parceiros) desce por cima. É o efeito mais marcante do site e cria a sensação de profundidade.
Hover — Project Cards
Zoom na imagem
Onde: Project Card — hover no card inteiro
.project-card { overflow: hidden; }

.project-card .image {
  transition: transform 0.4s ease,
              opacity 0.4s ease;
}

.project-card:hover .image {
  transform: scale(1.05);
  opacity: 0.85; /* aumenta levemente */
}
Cursor pointer
Onde: Project Card completo
.project-card {
  cursor: pointer;
}

/* Cards linkam para a página
   do projeto individual */
Hover — Navigation Links
Underline animado (slide from left)
Onde: Links de nav · "Let's Talk" CTA
.nav-link {
  position: relative;
  text-decoration: none;
}

.nav-link::after {
  content: '';
  position: absolute;
  bottom: -2px; left: 0;
  width: 100%; height: 2px;
  background: currentColor;
  transform: scaleX(0);
  transform-origin: left;
  transition: transform 0.25s ease;
}

.nav-link:hover::after {
  transform: scaleX(1);
}
O "Let's Talk" já tem o underline permanente (border-bottom: 2px). Para os outros links, o underline aparece no hover.
Animações de entrada (Scroll Reveal)
Fade-up ao entrar na viewport
Onde: Headings de seção, cards de projeto, itens de serviço, testimonials
/* CSS */
.reveal {
  opacity: 0;
  transform: translateY(24px);
  transition: opacity 0.6s ease-out,
              transform 0.6s ease-out;
}
.reveal.in-view {
  opacity: 1;
  transform: translateY(0);
}

/* JS — Intersection Observer */
const observer = new IntersectionObserver(
  (entries) => entries.forEach(e => {
    if (e.isIntersecting) e.target.classList.add('in-view');
  }),
  { threshold: 0.15 }
);
document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
Stagger (entrada em cascata)
Onde: Logos de parceiros, sub-itens de serviço, cards de testimonial
/* Adicionar delay progressivo */
.stagger-item:nth-child(1) { transition-delay: 0.0s; }
.stagger-item:nth-child(2) { transition-delay: 0.1s; }
.stagger-item:nth-child(3) { transition-delay: 0.2s; }
.stagger-item:nth-child(4) { transition-delay: 0.3s; }

/* Ou via JS dinamicamente */
items.forEach((el, i) => {
  el.style.transitionDelay = `${i * 0.1}s`;
});
Efeito da tipografia display (XXLarge)
Palavra decorativa de seção (ex: "PROJECTS")
Onde: Início de Section Projects — texto de 320px de fundo
.section-word-bg {
  font-family: 'Anton SC', sans-serif;
  font-size: 320px;
  text-transform: uppercase;
  line-height: 1;
  color: var(--color-text-default);  /* #0E1011 */
  pointer-events: none;
  user-select: none;
  /* NÃO tem opacidade reduzida —
     é preto cheio mas fica abaixo do subtítulo */
}
A palavra "PROJECTS" (320px) fica na mesma linha que o subtítulo descritivo (Inter 80px). A hierarquia visual funciona pela diferença de peso visual, não por opacidade.
